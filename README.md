No installation required just visit the link and start checking usernames!

## How to Use

1. **Visit the website** at the link above
2. **Enter usernames** in the text box (one per line)
3. **Adjust settings** (optional):
   - **Delay**: Time between batches in milliseconds (default: 200ms)
   - **Concurrent**: Number of simultaneous checks (default: 5)
4. **Click "Start Checking"** to begin
5. **View results** in real-time as they complete
6. **Export available usernames** by clicking "Export Available"
